FINAL v7.7.6

1. 쇼핑몰별 공식 API를 최종 기준으로 사용
- 샵모아 숫자에 맞추는 보정값이나 고정 숫자를 사용하지 않습니다.
- 쿠팡, 스마트스토어, 11번가, 롯데온이 보내는 현재 주문 상태를 직접 저장합니다.
- G마켓과 옥션은 연결 전까지 모든 집계에서 제외합니다.

2. 스마트스토어 선물하기 수락 대기 제외
- giftReceivingStatus=WAIT_FOR_RECEIVING은 실제 처리 주문이 아니므로 신규, 발송대기, 오늘 주문, 오늘 매출, 월 누계에서 제외합니다.
- 상대방이 수락하여 RECEIVED가 되면 그때부터 실제 주문 상태에 맞게 반영합니다.

3. 텔레그램 실제 상품 썸네일 보강
- 쿠팡: 판매자상품코드/등록상품ID 조회 후 상품 상세 이미지와 상세 HTML 이미지를 확인합니다.
- 스마트스토어: 채널상품, 원상품, 상품검색 응답의 대표 이미지를 순차 확인합니다.
- 11번가: 기존 정상 동작을 유지합니다.
- 롯데온: 판매자상품번호로 공식 상품 상세조회 API를 호출해 이미지를 확인합니다.
- 사진을 못 찾거나 전송 실패해도 주문 문자 알림은 반드시 전송합니다.

4. 무료한도 유지
- 상품 이미지 URL은 PC 로컬 캐시에 저장합니다.
- 주문 상태가 바뀌지 않으면 Firestore를 다시 쓰지 않습니다.

5. 기존 기능 포함
- 빠른 수집, 진행률, 오늘 분석 그래프, 주문일시, 앱 아이콘, 자동 재시작을 유지합니다.
