이번 파일은 기존 패치에 의존하지 않습니다.
사용자님이 올린 현재 실제 index.html을 직접 수정했습니다.

수정 내용
- dedupeInfo null 오류 제거
- statusGrid null 오류 방지
- Firebase 동적 로더 제거
- Firebase 10.14.1 고정 로딩
- Firestore 서버 확인을 기다리지 않고 실시간 리스너 즉시 시작
- 앱 재접속/화면 복귀 시 자동 재연결
- chrome-extension 주소 캐시 오류 제거
- 서비스워커 전체 교체 및 캐시 강제 갱신
- 앞으로 오류가 발생하면 상단에 실제 오류 메시지 표시

적용
1. 압축을 풉니다.
2. ACTUAL_V60_FIX 폴더를 기존 프로젝트 폴더 바로 아래에 둡니다.
3. install-actual-v60.cmd 실행
4. GitHub Desktop Summary:
   actual v60 firebase 화면중단 수정
5. Commit to main → Push origin
6. GitHub Pages 배포 완료 후 휴대폰 사이트 데이터 삭제
7. 사이트 다시 접속

PC 수집기와 backend 파일은 건드리지 않습니다.
