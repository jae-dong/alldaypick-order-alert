v7.7.19 당일전체·기간전체통계 마켓 상품행·금액 직접복구 안내

1. START_AGENT.cmd를 실행합니다.
2. 검은 창에서 쿠팡·스마트스토어·11번가·롯데온 동기화 완료를 기다립니다.
3. 프로젝트 폴더의 마켓직접검증_결과열기.cmd를 실행합니다.
4. backend\market-direct-audit.json에서 각 마켓의 마지막 직접 조회 시각과 결과를 확인합니다.

검증 기준
- 쿠팡: 주문 API, 취소·반품·교환·문의 API를 각각 직접 조회
- 스마트스토어: 주문/상품주문·클레임·문의 API를 직접 조회
- 11번가: 주문 API와 주문 상태/클레임 API를 직접 조회
- 롯데온: 출고지시 및 배송진행 API를 직접 조회하고 동일 상품행을 병합
- G마켓·옥션: API 승인 전이므로 집계와 직접검증에서 제외
- 샵모아 수치는 프로그램 계산에 사용하지 않으며 참고 비교만 합니다.

주의
- 파일에는 API 키, 비밀키, 주소, 전화번호, 구매자 이름을 기록하지 않습니다.
- 문의 API가 429 제한에 걸리면 직접검증 대기로 표시되고 오래된 캐시는 현재 문의 건수에서 제외됩니다.
- 스마트스토어 선물하기 미수락 주문은 신규주문·오늘 주문·매출·텔레그램에서 제외됩니다.
