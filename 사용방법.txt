올데이픽 주문알림 FINAL v6.0.0 — 완전 정리본

이 압축파일은 과거 버전, 백업폴더, apply/install 파일, node_modules,
잠금파일, 개인정보 설정을 모두 제외한 새 프로젝트 폴더입니다.

폴더 구성
- index.html
- app.js
- styles.css
- sw.js
- manifest.json
- icon.svg
- firestore.rules.txt
- backend/
- 1_기존설정_가져오기.cmd
- 2_PC수집기_시작.cmd

적용 방법
1. 현재 검은 수집기 창을 Ctrl+C로 종료합니다.
2. 현재 alldaypick-order-alert 폴더 이름을
   alldaypick-order-alert-OLD 로 변경합니다.
3. 이 압축을 GitHub 폴더에 풉니다.
4. 새 폴더 이름을 alldaypick-order-alert 로 변경합니다.
5. 새 폴더 안의 1_기존설정_가져오기.cmd를 실행합니다.
6. 기존 프로젝트 경로를 입력합니다.
   예: C:\Users\사용자\OneDrive\문서\GitHub\alldaypick-order-alert-OLD
7. GitHub Desktop에서 새 폴더를 열고 변경사항을 Commit/Push합니다.
8. 2_PC수집기_시작.cmd를 실행합니다.
9. 휴대폰에서 사이트 데이터를 삭제하고 다시 접속합니다.

정리된 내용
- 과거 수십 개 버전폴더 및 백업폴더 제외
- apply-v*.cmd/ps1, install-v*.cmd/ps1 제외
- node_modules 제외
- 오래된 잠금파일 제외
- 텔레그램 JSON BOM 자동 복구

- 시작 시 최근 14일 쿠팡 신규·발송대기 상태 정밀 정리

- 배송지시(DEPARTURE)를 발송대기로 분류
- 배송중·배송완료·구매확정은 처리알림에서 제외
