올데이픽 주문알림 v53 연결 유지 + 텔레그램 테스트

수정 내용
- 수집기가 1분마다 Firebase에 생존 신호 기록
- 새로고침 시 마지막 정상 쇼핑몰 연결 상태를 브라우저에 보존
- 지금 수집을 누르지 않아도 연결 상태 유지 표시
- 텔레그램 테스트 버튼 추가
- 버튼 클릭 → Firebase 명령 → 로컬 수집기 → 텔레그램 실제 전송
- 접힌 모바일 화면도 펼친 화면과 동일한 쇼핑몰별 표 사용
- 모바일 전용 요약 카드는 숨기고 동일한 열과 숫자 유지
- 화면 캐시 버전 강제 변경

적용
1. 압축 해제
2. apply-v53.cmd, apply-v53.ps1, v53-files 폴더를 프로젝트 폴더에 복사
3. PowerShell:
   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v53.ps1"
4. GitHub Desktop:
   v53 연결유지 텔레그램테스트
5. Commit to main → Push origin
6. 검은 수집기 창 Ctrl+C
7. backend에서 npm run agent
8. 휴대폰 사이트 데이터 한 번 삭제 후 다시 접속
9. 텔레그램 테스트 버튼 클릭

정상 로그
텔레그램 테스트 메시지 전송 성공
