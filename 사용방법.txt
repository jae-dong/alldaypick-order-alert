올데이픽 주문알림 v25 음성 볼륨 조절 삭제

삭제되는 기능

- 음성 볼륨 슬라이더
- 음성 소리 크기 조절 UI
- 저장되어 있던 음성 볼륨 설정값

유지되는 기능

- 휴대폰 푸시 알림
- 브라우저 기본 알림음
- 알림 권한
- 알림 테스트
- 쿠팡 주문 수집
- 신규주문·발송대기·배송상태 동기화
- 취소·반품·교환 자동수집
- 상태 카드 필터
- 쇼핑몰 연결현황 상단 배치
- 주문 통계 위치 개선

적용 방법

1. ZIP 압축을 풉니다.

2. 아래 두 파일을 기존 프로젝트 폴더에 복사합니다.

   apply-v25.cmd
   apply-v25.ps1

3. 프로젝트 폴더 주소창에 powershell 입력 후 실행합니다.

   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v25.ps1"

4. 성공 문구를 확인합니다.

   SUCCESS: v25 voice volume control removed.

5. GitHub Desktop에서

   Summary:
   v25 음성 볼륨 조절 삭제

   Commit to main
   Push origin

6. 배포 후 휴대폰 Chrome에서 사이트를 새로고침합니다.

   https://jae-dong.github.io/alldaypick-order-alert/

7. 예전 화면이 계속 보이면 Chrome 사이트 설정에서 데이터 삭제 후 다시 접속합니다.

이번 버전은 화면 정리 패치이므로 npm run agent를 다시 시작할 필요가 없습니다.
