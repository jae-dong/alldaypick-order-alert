올데이픽 주문알림 FINAL v7.0.1 — 실제 프로젝트 최종 정리본

이 폴더 하나만 사용합니다.
과거 버전 폴더, apply/install 패치, 중첩 프로젝트, 백업 파일을 모두 제거했습니다.

교체 방법
1. 실행 중인 검은 수집기 창에서 Ctrl+C를 누릅니다.
2. 기존 alldaypick-order-alert 폴더를 alldaypick-order-alert-OLD로 이름 변경합니다.
3. 이 압축 안의 alldaypick-order-alert 폴더를 같은 GitHub 위치에 놓습니다.
4. GitHub Desktop에서 File → Add local repository로 새 폴더를 선택합니다.
5. 이 프로젝트는 이미 최종 정리 커밋이 완료되어 있습니다.
6. GitHub Desktop 상단의 Push origin만 누릅니다.
7. PC수집기_시작.cmd를 실행합니다.
8. 휴대폰 사이트 데이터를 삭제한 뒤 다시 접속합니다.

구성
- 웹 운영파일 7개
- backend 운영파일
- 기존 Firebase/쇼핑몰 API 설정
- 기존 텔레그램 중복방지 기록
- 기존 GitHub 원격 연결
- PC수집기_시작.cmd
- PC수집기_강제종료.cmd
