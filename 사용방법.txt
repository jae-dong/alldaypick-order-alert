올데이픽 주문알림 v49 한눈에 보는 대시보드

변경 내용
- 상단 4개 핵심 지표로 정리
  1. 오늘 주문
  2. 오늘 매출
  3. 이번달 총주문
  4. 이번달 총매출
- 처리 알림을 핵심 지표 바로 아래 배치
- 쇼핑몰별 오늘 주문과 오늘 매출을 표 한 장으로 표시
- 신규·발송대기·취소·반품·교환도 함께 표시
- 쇼핑몰 행 클릭 시 해당 마켓 주문만 필터
- 마지막 업데이트 시간 표시
- 중복된 기존 쇼핑몰 요약 영역 숨김
- 기존 주문 상세·통계·텔레그램 알림 유지

적용 방법
1. 압축을 풉니다.
2. apply-v49.cmd와 apply-v49.ps1을 기존 프로젝트 폴더에 복사합니다.
3. 프로젝트 폴더 주소창에서 PowerShell을 엽니다.
4. 실행:
   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v49.ps1"
5. GitHub Desktop Summary:
   v49 한눈에 보는 대시보드
6. Commit to main → Push origin
7. 휴대폰에서 새로고침합니다.

이번 버전은 화면만 수정하므로 npm run agent 재시작은 필요 없습니다.
