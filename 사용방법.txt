올데이픽 주문알림 v48 텔레그램 전용

반영 내용
- 기존 웹푸시 발송 기능 제거
- 휴대폰 알림 권한·알림 테스트·푸시 등록 화면 제거
- 텔레그램만 주문 및 상태 알림 발송
- 텔레그램 실패 시 최대 3회 자동 재시도
- 쇼핑몰별 아이콘, 상품명, 옵션, 수량, 금액, 주문번호 표시
- 처리 알림을 쇼핑몰별 주문 현황 위에 배치
- 쇼핑몰별 건수는 오늘 주문만 표시
- 이번달 총주문과 이번달 총매출을 매월 1일부터 자동 누적
- Firebase는 주문 데이터와 통계 저장용으로 계속 사용

적용
1. 압축을 풉니다.
2. apply-v48.cmd, apply-v48.ps1, v48-files 폴더를 프로젝트 폴더에 복사합니다.
3. 프로젝트 폴더 주소창에서 PowerShell:
   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v48.ps1"
4. GitHub Desktop Summary:
   v48 텔레그램 전용 알림
5. Commit to main → Push origin
6. 기존 수집기 Ctrl+C
7. backend 폴더에서:
   npm run agent

npm install은 다시 하지 않아도 됩니다.

주의
- TELEGRAM_BOT_TOKEN과 TELEGRAM_CHAT_ID는 backend/.env.local에 그대로 둡니다.
- Firebase 서비스 계정은 주문 데이터 저장 때문에 삭제하면 안 됩니다.
