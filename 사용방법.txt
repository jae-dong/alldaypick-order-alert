올데이픽 주문알림 v20 쿠팡 전체 배송상태 동기화

추가 기능

- 신규주문
- 발송대기
- 배송지시
- 배송중
- 배송완료
- 직접배송

쿠팡 WING에서 상태가 변경되면 자동수집 시 Firestore 기존 주문이 업데이트되고,
웹의 상태 카드 숫자도 실시간으로 변경됩니다.

기존 카드 유지

- 주문취소
- 교환요청
- 반품요청
- 문의사항

이 네 항목은 현재 저장된 관련 문서가 있으면 숫자를 표시합니다.
쿠팡의 취소·반품·교환·문의 자동수집은 발주서 API와 별도 API이므로 추후 연결합니다.

동기화 범위

- 프로그램 시작 또는 웹 즉시수집: 최근 31일
- 10분 자동수집: 최근 7일
- 상태별 최대 50건을 초과하면 다음 페이지도 자동 조회

적용 방법

1. ZIP 압축을 풉니다.

2. 아래 3개를 기존 프로젝트 폴더 안에 복사합니다.

   apply-v20.cmd
   apply-v20.ps1
   v20-files 폴더

3. 프로젝트 폴더 주소창에 powershell 입력 후 실행:

   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v20.ps1"

4. 성공 문구 확인:

   SUCCESS: v20 full Coupang status sync installed.

5. GitHub Desktop:

   Summary:
   v20 쿠팡 전체 배송상태 동기화

   Commit to main
   Push origin

6. 기존 npm run agent 창을 Ctrl+C로 종료합니다.

7. backend 폴더에서 다시 실행:

   npm run agent

정상 로그 예시

동기화 완료: 신규 2, 발송대기 5, 배송지시 3,
배송중 8, 배송완료 12, 직접배송 0

주의

현재 실제 API 연결 쇼핑몰은 쿠팡입니다.
다른 쇼핑몰은 해당 쇼핑몰 API 키와 주문 API 연결 후 같은 구조로 추가합니다.
