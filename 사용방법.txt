올데이픽 주문알림 v23 쿠팡 취소·반품·교환 자동수집

추가 기능

- 주문취소 자동수집
- 반품요청 자동수집
- 교환요청 자동수집
- Firestore orders 문서에 저장
- v22 상태판의 주문취소·반품요청·교환요청 숫자 자동 반영
- 신규 취소·반품·교환 발생 시 휴대폰 푸시
- HTTP 429 자동대기 유지

호출 제한 안정화

10분마다 세 가지 중 하나만 순환 조회합니다.

1회차 주문취소
2회차 반품요청
3회차 교환요청

따라서 각 항목은 약 30분마다 갱신됩니다.

적용 방법

1. ZIP 압축을 풉니다.
2. apply-v23.cmd, apply-v23.ps1, v23-files 폴더를 프로젝트 폴더에 복사합니다.
3. 프로젝트 폴더 주소창에 powershell 입력 후 실행:

   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v23.ps1"

4. GitHub Desktop:

   Summary:
   v23 취소 반품 교환 자동수집

   Commit to main
   Push origin

5. 기존 npm run agent 창 Ctrl+C
6. backend 폴더에서 npm run agent

정상 로그 예시

CS동기화 완료: 주문취소 발견 2, 신규 1, 상태변경 0

참고

- 결제완료 단계 취소와 출고중지 요청을 주문취소로 표시합니다.
- 반품 접수 상태를 반품요청으로 표시합니다.
- 교환 목록의 접수·진행·완료 상태를 모두 저장하되 상태판에서는 교환요청 항목에 합산합니다.
- 문의사항 자동수집은 쿠팡 고객문의 API를 연결하는 다음 단계입니다.
