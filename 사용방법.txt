올데이픽 주문알림 v30 전체 정리본

이번 버전은 기존 index.html에 누적된 패치를 모두 제거하고 화면과 코드를 새로 구성합니다.

반영 내용
- 쇼핑몰 연결 현황을 화면 최상단 배치
- 신규주문·발송대기·배송지시·배송중·배송완료·직접배송
- 주문취소·교환요청·반품요청·문의사항 상태 카드
- 상태 카드를 누르면 해당 주문만 즉시 표시
- 주문 통계 탭을 누르면 화면 바로 위에서 통계 표시
- 모바일 오른쪽 여백과 좌우 스크롤 제거
- 휴대폰 푸시 등록상태 자동 복원
- 음성 안내와 음성 볼륨 조절 기능 완전 제거
- 시작·즉시수집 시 취소·반품·교환 전체 확인
- 기존 Firebase 설정 자동 보존
- backend/.env.local 쿠팡 키 자동 보존
- 설치 전 파일 자동 백업

설치
1. 압축을 풉니다.
2. 압축 안의 install-v30.cmd, install-v30.ps1, v30-project 폴더를
   기존 문서\GitHub\alldaypick-order-alert 폴더에 복사합니다.
3. 프로젝트 폴더 주소창에 powershell 입력 후 아래 실행:
   powershell -NoProfile -ExecutionPolicy Bypass -File ".\install-v30.ps1"
4. SUCCESS 문구 확인
5. GitHub Desktop에서 Summary:
   v30 전체 구조 정리
6. Commit to main → Push origin
7. 기존 npm run agent 창 Ctrl+C
8. backend 폴더에서 npm install 후 npm run agent
   node_modules가 있으면 npm run agent만 실행하면 됩니다.
9. 휴대폰 Chrome에서 사이트 데이터 삭제 후 재접속
