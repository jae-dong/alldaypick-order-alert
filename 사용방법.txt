올데이픽 주문알림 v19 상태 자동동기화 통합본

이 버전에는 아직 적용하지 않은 v18 모바일 화면 수정도 함께 포함되어 있습니다.

추가 기능

1. 신규주문 옆에 발송대기 카드가 추가됩니다.
2. 쿠팡 WING에서 신규주문을 확인해 상품준비중으로 변경하면,
   다음 수집 시 웹 주문 상태가 발송대기로 자동 변경됩니다.
3. 기존 주문 문서를 새로 중복 생성하지 않고 상태만 업데이트합니다.
4. 10분 자동수집과 웹 즉시수집 모두 신규주문과 발송대기를 확인합니다.
5. 중복 백그라운드 푸시 영역을 숨깁니다.
6. 모바일 오른쪽 여백과 좌우 스크롤 문제를 함께 수정합니다.

적용 방법

1. ZIP 압축을 풉니다.

2. 아래 3개를 기존 프로젝트 폴더 안에 복사합니다.

   apply-v19.cmd
   apply-v19.ps1
   v19-files 폴더

   위치:
   문서\GitHub\alldaypick-order-alert

3. CMD가 깨졌던 PC에서는 프로젝트 폴더 주소창에 powershell을 입력한 뒤
   아래 명령으로 실행하는 것을 권장합니다.

   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v19.ps1"

4. 성공 문구를 확인합니다.

   SUCCESS: v19 Coupang status sync installed.

5. GitHub Desktop에서

   Summary:
   v19 신규주문 발송대기 자동동기화

   Commit to main
   Push origin

6. 실행 중인 npm run agent 창에서 Ctrl+C를 누릅니다.

7. backend 폴더에서 다시 실행합니다.

   npm run agent

정상 로그 예시

신규주문 3, 발송대기 5, 신규저장 0, 상태변경 1

중요

- 이번 버전에서 실제 상태 자동연동되는 쇼핑몰은 현재 API가 연결된 쿠팡입니다.
- 스마트스토어, 11번가, G마켓, 옥션, 롯데온은 각각 API 키와 주문 API를 연결한 뒤
  동일한 방식으로 상태 자동동기화를 추가합니다.
- 취소, 반품, 교환, 문의는 쿠팡에서도 별도 API이므로 다음 단계에서 순차 추가합니다.
