올데이픽 주문알림 FINAL v7.5.0 FREE TIER 사용방법

1. GitHub Desktop에서 Repository > Show in Explorer를 눌러 실제 연결 폴더를 엽니다.
2. 새 버전의 내용물만 그 폴더에 덮어씁니다.
3. backend/.env.local과 .git 폴더는 삭제하지 않습니다.
4. GitHub Desktop에서 Commit to main > Push origin을 누릅니다.
5. 웹 화면이 FINAL v7.5.0으로 바뀐 것을 확인합니다.
6. 기존 검은 수집기 창을 닫고 STOP_AGENT.cmd를 한 번 실행합니다.
7. 최종 GitHub 연결 폴더의 START_AGENT.cmd를 실행합니다.
8. 검은 창은 계속 켜둡니다.

무료 한도 운영 주의
- FAST_POLL_MINUTES=10, POLL_INTERVAL_MINUTES=30을 유지하세요.
- RUN_LEGACY_MIGRATION=0, ENABLE_DIAGNOSTICS=0을 유지하세요.
- 휴대폰에서 캐시를 반복 삭제하거나 페이지를 계속 새로고침하지 마세요.
