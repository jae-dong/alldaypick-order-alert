올데이픽 주문알림 v33

추가 기능

1. 스마트스토어 신규주문 휴대폰 푸시
2. 쇼핑몰별 주문 현황 카드 클릭 필터
3. 쿠팡 또는 스마트스토어 주문만 즉시 보기
4. 구매확정 상태 카드 추가
5. 주문 상세정보 확대
   - 옵션
   - 연락처
   - 주소
   - 배송메모
   - 택배사
   - 운송장번호
6. 연락처 복사 버튼 추가
7. 주문목록 40건씩 페이지 나누기
8. 검색·쇼핑몰·상태 필터 변경 시 첫 페이지로 이동
9. 스마트스토어 구매확정 상태 자동 반영

적용 방법

1. ZIP 압축을 풉니다.
2. apply-v33.cmd, apply-v33.ps1, v33-files 폴더를
   기존 alldaypick-order-alert 폴더에 복사합니다.
3. 프로젝트 폴더 주소창에서 powershell 실행:

   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v33.ps1"

4. backend 폴더에서 실행:

   npm install

5. GitHub Desktop:

   Summary:
   v33 스마트스토어 푸시 및 주문필터

   Commit to main
   Push origin

6. 기존 npm run agent 창 Ctrl+C
7. backend 폴더에서:

   npm run agent

스마트스토어 신규주문이 새로 저장되면
쿠팡과 같은 방식으로 휴대폰 푸시가 전송됩니다.
