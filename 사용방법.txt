올데이픽 주문알림 v15 안전 적용본

이 버전은 현재 정상 작동 중인 index.html과 Firebase API 키를 바꾸지 않습니다.
푸시 기능에 필요한 backend 파일과 sw.js만 교체합니다.

적용 방법

1. ZIP 압축을 풉니다.

2. 압축을 푼 폴더 안의 아래 3개를 기존 프로젝트 폴더 안에 복사합니다.

   apply-v15.cmd
   apply-v15.ps1
   v15-files 폴더

   기존 프로젝트 폴더:
   문서\GitHub\alldaypick-order-alert

3. alldaypick-order-alert 폴더 안에 들어간 apply-v15.cmd를 더블클릭합니다.

4. 아래 문구가 나오면 성공입니다.

   SUCCESS: v15 push files applied.
   Firebase API configuration was preserved.

5. GitHub Desktop에서 변경된 파일을 Commit 후 Push origin 합니다.

6. 실행 중인 로컬 수집기 창에서 Ctrl+C를 누릅니다.

7. backend\start-local-agent.cmd를 다시 실행합니다.

8. 로컬 수집기 화면 마지막에 다음 문구가 나오면 정상입니다.

   휴대폰 푸시 대기

휴대폰 등록

1. 실제 사이트 접속
   https://jae-dong.github.io/alldaypick-order-alert/

2. 백그라운드 푸시 영역에서 VAPID 공개키 입력

3. 이 휴대폰 푸시 등록 누르기

4. 알림 권한 허용

5. 본인 휴대폰과 배우자 휴대폰에서 각각 한 번씩 등록

주의

- VAPID 공개키는 Firebase 웹 푸시 인증서에서 발급합니다.
- Secret Key나 Firebase 서비스 계정 키는 휴대폰에 입력하지 않습니다.
- 집 PC의 로컬 수집기 창이 켜져 있어야 주문 수집과 푸시 발송이 됩니다.
