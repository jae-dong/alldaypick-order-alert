올데이픽 주문알림 v34 11번가 연동 준비

이번 버전

- backend/.env.local에 11번가 Open API 키 입력 자리 추가
- 11번가 API 키 등록 여부를 자동 확인
- 웹의 쇼핑몰 연결 현황에 아래 상태 표시

  미등록:
  Open API 키 등록 필요

  등록:
  API 키 등록 완료 · 주문 API 연결 대기

- 기존 쿠팡·스마트스토어 동기화와 푸시는 그대로 유지

적용

1. ZIP 압축을 풉니다.
2. apply-v34.cmd, apply-v34.ps1, v34-files 폴더를 프로젝트 폴더에 복사합니다.
3. 프로젝트 폴더 주소창에 powershell 입력 후 실행:

   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v34.ps1"

4. backend/.env.local 마지막에 추가:

   ELEVENST_API_KEY=실제_11번가_Open_API_Key
   ELEVENST_SELLER_ID=판매자아이디

5. GitHub Desktop:
   v34 11번가 연동 준비
   Commit → Push origin

6. 기존 npm run agent 창 Ctrl+C
7. backend 폴더에서:

   npm run agent

중요

11번가 주문 API의 실제 호출 코드는 발급받은 키의 사용 권한과
Open API Center에서 제공되는 주문 API 문서를 확인한 뒤 연결합니다.
키와 판매자 아이디는 채팅에 보내지 마세요.
