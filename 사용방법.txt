올데이픽 주문알림 v31

화면 개선
- 쇼핑몰별 주문 현황을 쇼핑몰 연결 현황 바로 아래로 이동
- 상태 카드 선택 색상 구분
- 주문 행 클릭 시 상세정보 팝업
- 주문번호·구매자·상품명·운송장번호 복사
- 쇼핑몰 최근 확인시간을 방금 전·몇 분 전 형식으로 표시

다른 마켓 연동 시작
- 스마트스토어 OAuth 인증 모듈 추가
- 스마트스토어 변경 주문 조회 및 상세 주문 조회 구조 추가
- 키 미등록 시 'API 키 등록 필요'
- 키 등록 후 주문·배송·취소·반품·교환 상태를 Firestore에 저장
- 11번가·G마켓·옥션·롯데온은 다음 순서로 연결

적용
1. 압축을 풉니다.
2. apply-v31.cmd, apply-v31.ps1, v31-files 폴더를 프로젝트 폴더에 복사합니다.
3. 프로젝트 폴더 주소창에서 powershell 실행:
   powershell -NoProfile -ExecutionPolicy Bypass -File ".\apply-v31.ps1"
4. backend 폴더에서 npm install
5. GitHub Desktop:
   v31 주문상세 및 스마트스토어 연동시작
   Commit → Push origin
6. 기존 에이전트 Ctrl+C
7. backend 폴더에서 npm run agent

스마트스토어 연결
backend\.env.local 마지막에 아래 두 줄을 추가합니다.

NAVER_CLIENT_ID=커머스API 애플리케이션 ID
NAVER_CLIENT_SECRET=커머스API 애플리케이션 시크릿

커머스API센터에서 애플리케이션을 등록하고 권한을 받은 키여야 합니다.
