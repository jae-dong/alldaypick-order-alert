올데이픽 주문알림 FINAL v7.4.0 REVIEWED

핵심 변경
- 주문 진행 상태와 취소/반품/교환/문의를 별도 문서 흐름으로 분리
- 주문번호+상품 줄 기준 최신 상태만 집계
- 신규 -> 발송대기 -> 배송중/배송완료 전환 반영
- 완료/철회/거부/답변완료 사건 자동 비활성화
- 과거 혼합 상태 문서 1회 정리
- 스마트스토어 변경내역 페이지 커서 처리 수정
- 스마트스토어 상품문의 필수 기간·1번 페이지·100건 페이지 처리
- 스마트스토어 고객문의 inquiryNo/등록일시/문의내용 필드 처리
- 쿠팡 상품문의와 고객센터 미처리 문의 조회 및 페이지 처리
- 쿠팡 429/502/503/504와 응답 지연 재시도
- 11번가 주문 문서와 클레임 문서 분리 및 완료 목록 정리
- PC 수집기 생존신호 1분 갱신
- 중복 PC 수집기 실행 방지
- G마켓/옥션 미연결 집계 제외

검토 범위
- 브라우저 상태 엔진 단위 테스트
- 백엔드 상태 판정 단위 테스트
- Firestore 저장/재활성화/상태정리 단위 테스트
- API 소스 계약 정적 테스트
- 스마트스토어 문의 변환 단위 테스트
- 전체 JavaScript 문법 검사
- ZIP 무결성 및 SHA-256 생성

제한
- 실제 판매자 계정 API 응답과 Firestore 운영 데이터는 이 실행 환경에서 직접 호출하지 않았습니다.
- 첫 배포 후 실제 수치 검증은 반드시 연결된 4개 마켓 관리자 화면과 비교해야 합니다.
