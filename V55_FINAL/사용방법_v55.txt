올데이픽 주문알림 CLEAN v55

- 발송대기로 처음 발견된 주문도 텔레그램 전송
- 텔레그램 15초 제한, 3회 재시도
- 쿠팡 지연이 다른 마켓과 텔레그램 테스트를 막지 않음
- PC 수집기 생존신호 즉시 시작
- 주문 목록은 최근 31일 전체 주문 기본 표시
- 오늘 분석 항상 표시
- 지금 수집은 이번달 1일부터 정밀 동기화
- 각 마켓 독립 실행, 한 마켓 실패해도 나머지는 계속

각 쇼핑몰 API는 사용자 PC 허용 IP로만 호출되므로 정밀동기화는 사용자 PC에서 실행됩니다.

적용:
1. 압축 해제
2. V55_FINAL 폴더를 프로젝트 바로 아래에 둡니다.
3. install-v55.cmd 실행
4. GitHub Desktop: v55 정밀동기화 텔레그램수정
5. Commit → Push
6. 검은 창 Ctrl+C
7. backend에서 npm install
8. npm run agent
9. 텔레그램 테스트
10. 지금 수집 눌러 이번달 정밀 동기화
