올데이픽 주문알림 v59 화면 중단 긴급수정

확인된 실제 오류
- renderStatus 함수가 존재하지 않는 dedupeInfo 요소에 textContent를 입력
- 이 오류로 전체 JavaScript 실행이 중단
- Firebase, PC 수집기, 텔레그램 상태와 주문 데이터가 화면에 반영되지 않음

함께 수정한 오류
- 서비스워커가 chrome-extension:// 주소를 캐시에 넣으려던 오류
- Failed to convert value to Response
- Cache.put unsupported request scheme

적용
1. 압축을 풉니다.
2. apply-v59.cmd와 apply-v59.ps1을 기존 프로젝트 폴더에 복사합니다.
3. apply-v59.cmd 실행
4. GitHub Desktop Summary:
   v59 화면중단 긴급수정
5. Commit to main → Push origin
6. GitHub Pages 배포 완료 후 휴대폰 사이트 데이터 삭제
7. 다시 접속

이번 수정은 화면과 서비스워커만 변경합니다.
검은 PC 수집기는 재시작할 필요 없습니다.
