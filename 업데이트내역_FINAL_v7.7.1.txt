올데이픽 주문알림 FINAL v7.7.1 빠른수집·교환정리·텔레그램 썸네일

1. 지금 수집 속도 개선
- 연결된 4개 마켓 주문을 병렬 확인
- 쿠팡 빠른 단계는 최근 7일/최대 4페이지의 현재 신규·발송대기부터 우선 반영
- 버튼 100% 완료 후 쿠팡 31일 현재상태 정밀 보정은 백그라운드에서 자동 진행
- 스마트스토어 문의 API는 지금 수집 과정에서 기다리지 않고 캐시 표시 후 정기 조회
- 11번가 불필요 대기시간 단축
- 롯데온 지금 수집 시 별도 연결 테스트 1회를 생략하고 주문조회로 바로 확인

2. 쿠팡 완료 교환 강제 정리
- 쿠팡 교환 API가 완전 조회된 경우 Firestore의 활성 교환 문서를 직접 대조
- 현재 미처리 교환 목록에 없는 과거 교환 문서는 완료로 전환
- 로컬 캐시 문서 ID가 달라 남던 완료 교환도 제거
- API 조회가 불완전하거나 실패한 경우에는 안전을 위해 자동 제거하지 않음

3. 텔레그램 신규주문 썸네일
- 신규주문 알림에 상품 썸네일 사진과 주문정보를 함께 전송
- 주문일시·상품명·옵션·수량·금액·주문번호 표시 유지
- 쿠팡은 등록상품 조회 API, 스마트스토어는 채널상품 조회 API를 사용해 대표이미지 확인
- 11번가·롯데온은 주문 응답 이미지 또는 공개 상품페이지 대표이미지를 확인
- 이미지가 없거나 텔레그램 사진 전송이 실패하면 기존 텍스트 알림으로 자동 재전송
- 이미지 결과는 PC 로컬 캐시에 저장해 같은 상품을 반복 조회하지 않음

4. 기존 기능 유지
- 오늘 주문·매출 및 현재 진행 주문 집계
- G마켓·옥션 제외
- 무료 Firestore 한도 최적화
- 수집 진행률 표시
- 밝은 대시보드와 오늘분석 그래프
