올데이픽 주문알림 FINAL v7.7.3 텔레그램 실제 상품 썸네일 전송 수정

1. 쇼핑몰 이미지 URL을 텔레그램 서버에 넘기는 방식 대신 PC 수집기가 이미지를 직접 다운로드합니다.
2. 다운로드한 실제 이미지 파일을 multipart/form-data 방식으로 텔레그램에 업로드합니다.
3. 쇼핑몰 CDN이 외부 원격 접근을 차단해도 PC에서 받을 수 있으면 사진 알림이 전송됩니다.
4. 파일 업로드 실패 시 이미지 URL 방식으로 한 번 더 시도하고, 최종 실패 시 텍스트 알림으로 전환합니다.
5. 쿠팡은 sellerProductId가 없을 때 판매자 상품코드로 상품정보를 다시 찾아 대표 이미지를 조회합니다.
6. 스마트스토어는 주문의 채널 상품번호로 채널 상품 API를 조회해 대표 이미지를 가져옵니다.
7. 11번가와 롯데온은 주문 응답 이미지와 실제 상품페이지 대표 이미지를 순서대로 확인합니다.
8. 이전 버전의 '이미지 없음' 캐시는 폐기하고 새 캐시로 다시 조회합니다.
9. 검은 창에 썸네일 주소 없음, 파일 업로드 성공, 실패 원인을 표시합니다.
10. v7.7.2의 자동 재시작, 빠른 수집, 디자인, 상태 로직, 무료한도 최적화는 그대로 유지합니다.
