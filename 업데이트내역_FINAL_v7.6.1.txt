올데이픽 주문알림 FINAL v7.6.1

1. 11번가 발송대기 누락 복구
- 과거 부분 응답으로 잘못 비활성화된 신규/발송대기 문서도 다시 상태 확인합니다.
- 여러 주문번호 조회에서 일부만 반환되면 누락 주문번호를 개별 재조회합니다.
- 모든 주문번호가 확인된 경우에만 미처리 목록 정리를 수행합니다.
- 배송중/배송완료로 확인된 주문은 발송대기에서 제거합니다.

2. 쿠팡 완료 교환 제거
- 교환 상태 RECEIPT/PROGRESS만 미완료로 표시합니다.
- SUCCESS/REJECT/CANCEL은 완료·불가·철회로 처리하여 화면에서 제거합니다.

3. 스마트스토어 문의 429 보호
- 문의 API는 주문 API와 분리해 기본 60분 간격으로 조회합니다.
- HTTP 429 발생 시 기본 120분 동안 재호출하지 않고 기존 미답변 문의 캐시를 표시합니다.
- 주문 수집은 문의 API 제한과 무관하게 계속 진행합니다.

4. 유지 사항
- 오늘 주문/매출 계산 로직 유지
- 신규→발송대기→배송중 상태 이동 유지
- 미완료 반품/교환/문의만 표시
- G마켓/옥션 미연결 제외
- Firestore 무료한도 로컬 캐시 최적화 유지
