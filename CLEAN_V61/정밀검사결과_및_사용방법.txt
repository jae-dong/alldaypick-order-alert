- 실제 현재 backend 모듈을 그대로 사용

적용
1. CLEAN_V61 폴더를 기존 프로젝트 폴더 바로 아래에 둡니다.
2. install-clean-v61.cmd 실행
3. GitHub Desktop Summary: clean v61 전체재구성
4. Commit to main → Push origin
5. 검은 창 Ctrl+C
6. 기존 프로젝트 backend에서 npm install
7. npm run agent
8. 휴대폰에서 사이트 데이터 삭제 후 다시 접속
