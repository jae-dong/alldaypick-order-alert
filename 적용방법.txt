올데이픽 주문알림 v51 전체 교체본

이번 파일은 누적 패치를 제거하고 실제 프로젝트를 기준으로 index.html을 처음부터 다시 정리한 버전입니다.

반영 내용
- 쇼핑몰 연결 현황 최상단
- 오늘 주문 / 오늘 매출 / 이번달 총주문 / 이번달 총매출
- 처리 알림을 쇼핑몰별 오늘 현황 위에 배치
- 쇼핑몰별 주문과 매출은 오늘 기준
- 쇼핑몰 행과 처리 상태 클릭 필터
- 주문 상세 및 복사 기능
- 주문 검색, 확인 처리, 통계, 직접 추가
- 웹푸시 화면 완전 제거, 텔레그램 안내만 유지
- 서비스워커 캐시를 v51로 교체하고 화면은 네트워크 우선 로딩

적용 방법
1. 기존 프로젝트 폴더를 백업합니다.
2. 이 ZIP의 모든 파일과 폴더를 기존 alldaypick-order-alert 폴더에 덮어씁니다.
3. backend/.env.local은 ZIP에 포함하지 않았으므로 기존 파일이 그대로 남습니다.
4. backend/node_modules도 포함하지 않았으므로 기존 설치가 그대로 유지됩니다.
5. GitHub Desktop에서 변경사항 확인
6. Summary: v51 전체 화면 재구성
7. Commit to main → Push origin
8. 휴대폰에서 사이트를 다시 엽니다. 이전 화면이면 Chrome 사이트 데이터 한 번 삭제 후 재접속합니다.

수집기 재시작은 필요 없지만, backend 파일도 함께 덮어쓴 경우 안전하게 npm run agent를 재시작해도 됩니다.
