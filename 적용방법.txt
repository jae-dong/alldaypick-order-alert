적용
1. ZIP 압축을 풉니다.
2. 기존 프로젝트 폴더에 전체 파일을 덮어씁니다.
3. backend/.env.local은 삭제하거나 덮어쓰지 마세요.
4. GitHub Desktop Summary:
   v60 클라우드 강제복구
5. Commit to main → Push origin
6. 검은 창 Ctrl+C
7. backend 폴더에서:
   npm install
   npm run agent
8. 휴대폰에서 사이트를 다시 엽니다.
9. 첫 접속 때 자동으로 캐시를 지우고 한 번 새로고침됩니다.
