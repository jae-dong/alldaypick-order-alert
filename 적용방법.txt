적용 방법
1. ZIP 압축을 풉니다.
2. 기존 프로젝트 폴더에 전체 파일을 덮어씁니다.
3. backend/.env.local은 삭제하거나 덮어쓰지 마세요.
4. GitHub Desktop Summary:
   v57 현재기준 및 미처리누적
5. Commit to main → Push origin
6. 검은 창 Ctrl+C
7. backend 폴더에서:
   npm install
   npm run agent
8. 휴대폰 사이트 새로고침

기존 화면이면 휴대폰 사이트 데이터를 한 번 삭제하고 다시 접속하세요.
