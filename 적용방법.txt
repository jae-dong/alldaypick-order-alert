올데이픽 주문관리 PRO v52

추가 기능
- 시간대별 오늘 주문 그래프
- 쇼핑몰별 오늘 매출 비중
- 오늘 판매 TOP 10 상품
- 오늘 출고 필요 주문 수
- 중요 주문 별표 표시
- 주문별 메모 저장
- 처리 완료 체크
- 미처리·중요·처리완료 필터
- 중요 주문은 목록 상단 우선 표시
- 매일 오전 9시 텔레그램 오늘 현황
- 매일 밤 11시 텔레그램 마감 요약

적용 방법
1. 기존 프로젝트 폴더를 백업합니다.
2. 이 ZIP 안의 모든 파일과 폴더를 기존 프로젝트에 덮어씁니다.
3. backend/.env.local은 ZIP에 포함하지 않았으므로 기존 키와 텔레그램 설정은 유지됩니다.
4. GitHub Desktop Summary:
   v52 주문관리 PRO 기능
5. Commit to main → Push origin
6. 기존 npm run agent 창에서 Ctrl+C
7. backend 폴더에서 npm run agent
8. 휴대폰에서 사이트를 새로고침합니다.

주의
- 일일 요약은 PC 수집기가 켜져 있을 때 전송됩니다.
- 순이익 계산은 상품별 매입가 데이터 연결 후 다음 단계에서 추가합니다.
