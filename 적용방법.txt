올데이픽 주문관리 PRO v54 주문현황 간소화

변경 내용
- 주문현황 목록에는 신규주문과 발송대기만 표시
- 배송중·배송완료·구매확정은 목록에서 제외
- 취소·반품·교환·문의는 처리 알림에서 최근 7일 기준으로 확인
- 처리 알림의 취소·반품·교환·문의를 누르면 해당 내역만 목록에 표시
- 신규주문·발송대기 탭을 누르면 해당 주문만 표시
- 오늘 주문·오늘 매출·이번달 총주문·이번달 총매출 집계는 기존대로 유지

적용 방법
1. 기존 프로젝트를 백업합니다.
2. ZIP 안의 모든 파일과 폴더를 기존 프로젝트에 덮어씁니다.
3. backend/.env.local은 포함하지 않았으므로 기존 설정은 유지됩니다.
4. GitHub Desktop Summary:
   v54 주문현황 간소화
5. Commit to main → Push origin
6. 기존 수집기 창 Ctrl+C
7. backend 폴더에서 npm run agent
8. 휴대폰에서 새로고침합니다.
