올데이픽 주문알림 v61 모바일 압축화면

모바일 개선
- 상단 핵심 지표를 2칸으로 압축
- 카드 높이와 글자 간격 축소
- 처리 알림을 3칸 x 2줄로 표시
- 쇼핑몰별 현황은 세로 카드로 바뀌지 않고 항상 표 형태 유지
- 첫 번째 쇼핑몰명 열은 고정
- 숫자 열은 좌우 스크롤
- 오늘 분석 영역은 모바일에서 숨겨 핵심 현황을 먼저 표시
- 주문 목록과 검색 영역도 모바일 크기에 맞게 축소
- 폴드폰 접힘/펼침 모두 같은 표 구조 유지

적용
1. ZIP 압축을 풉니다.
2. 기존 프로젝트 폴더에 전체 파일을 덮어씁니다.
3. backend/.env.local은 삭제하거나 덮어쓰지 마세요.
4. GitHub Desktop Summary:
   v61 모바일 압축화면
5. Commit to main → Push origin
6. 휴대폰 사이트를 다시 엽니다.
7. 이전 화면이면 사이트 데이터를 한 번 삭제하고 다시 접속하세요.

화면 수정 버전이므로 npm run agent 재시작은 필수가 아닙니다.
