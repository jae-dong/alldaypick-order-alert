FINAL v7.7.13 적용 방법

1. 실행 중인 검은색 PC 수집기 창을 닫습니다.
2. 압축파일을 새 폴더에 압축 해제합니다.
3. GitHub Desktop에서 alldaypick-order-alert 저장소를 선택합니다.
4. Repository → Show in Explorer를 눌러 기존 프로젝트 폴더를 엽니다.
5. 압축 해제 폴더 안의 모든 파일을 기존 프로젝트 폴더에 복사하고 모두 덮어씁니다.
6. .git, backend\.env.local, backend\firebase-service-account.json은 삭제하지 않습니다.
7. GitHub Desktop Summary에 아래 문구를 입력합니다.
   FINAL v7.7.13 스마트스토어·롯데온 상태정합성 보강
8. Commit to main → Push origin을 누릅니다.
9. Render에서 최신 커밋 배포 성공을 확인합니다.
10. 기존 프로젝트 폴더의 START_AGENT.cmd를 실행합니다.
11. 검은 창에 아래 문구가 표시되는지 확인합니다.
    Starting ALLDAYPICK order agent v7.7.13 SMARTSTORE LOTTEON STATE FIX...
12. 시작 동기화 후 스마트스토어 로그의 오늘전체대조와 롯데온 로그의 잔존정리 값을 확인합니다.
13. 검은 창을 닫지 않고 계속 켜 둡니다.
14. 웹앱을 Ctrl+F5로 강력 새로고침합니다.
15. 웹 상단 FINAL v7.7.13을 확인합니다.
16. G마켓·옥션은 미연동·집계 제외로 유지되는지 확인합니다.
