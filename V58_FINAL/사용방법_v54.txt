- 쇼핑몰별 주문 현황을 7열로 압축해 일반 스마트폰과 폴드폰 모두 가로 스크롤 제거
- 주문 수와 매출을 한 칸에 함께 표시

- 쿠팡/스마트스토어/11번가/롯데온 API 호출에 30초 제한
- 쿠팡 첫 수집이 늦어도 PC 수집기 생존 신호는 1분마다 계속 기록
- 텔레그램 테스트는 시작 수집 완료를 기다리지 않고 즉시 사용 가능
- 기존 backend/.env.local 자동 보존

적용
1. 압축 해제
2. V54_FINAL 폴더를 기존 프로젝트 폴더 바로 아래에 둡니다.
3. install-v54.cmd 실행
4. GitHub Desktop Summary: v54 모바일표 수집기 안정화
5. Commit to main → Push origin
6. 검은 창 Ctrl+C
7. backend에서 npm install
8. npm run agent
9. 휴대폰 사이트 데이터 한 번 삭제 후 재접속
10. 텔레그램 테스트 버튼 확인
