FINAL v7.2.1 새 프로젝트 설치 순서

1. ZIP 압축을 새 폴더에 완전히 풉니다.
2. START_AGENT.cmd를 실행하고 검은 창을 계속 켜둡니다.
3. GitHub Desktop에서 File > New repository를 누릅니다.
4. Name: alldaypick-order-alert-v721
5. Local path: 압축을 푼 폴더의 바로 윗폴더
6. Git ignore / License는 None으로 두고 Create repository를 누릅니다.
7. 변경 파일 전체가 보이면 Summary에 FINAL v7.2.1 새 프로젝트 등록 입력
8. Commit to main을 누릅니다.
9. Publish repository를 눌러 GitHub에 새 저장소로 올립니다.
10. Render에서 기존 서비스의 연결 저장소를 새 저장소로 바꾸거나 새 Web Service를 연결합니다.
11. Render 배포가 Live가 되면 웹앱을 완전히 종료 후 다시 열고 지금 수집을 누릅니다.

주의: 이 패키지에는 .git 기록이 없으므로 기존 원격 저장소와 병합 충돌이 발생하지 않습니다.
