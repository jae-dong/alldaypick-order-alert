기존 GitHub Pages 프로젝트를 계속 사용하는 설치 순서

새 저장소를 만들지 않습니다.

1. GitHub Desktop에서 저장소 alldaypick-order-alert를 선택합니다.
2. Repository > Show in Explorer로 실제 폴더를 엽니다.
3. 업데이트 ZIP 안쪽 alldaypick-order-alert 폴더의 내용물만 덮어씁니다.
4. Commit to main 후 Push origin을 누릅니다.
5. 기존 주소 https://jae-dong.github.io/alldaypick-order-alert/ 에 자동 반영됩니다.
6. 최종 프로젝트 폴더의 START_AGENT.cmd를 실행합니다.
