올데이픽 주문알림 v13

추가 기능
1. 10분 자동 쿠팡 주문수집
2. 웹 새로고침 시 즉시 수집 요청
3. '지금 수집' 버튼
4. 같은 주문 중복 저장 방지
5. 즉시수집 진행상태 표시

중요
- 웹 새로고침 즉시수집은 집 PC에서 local-agent가 켜져 있어야 합니다.
- GitHub Runner의 'Listening for Jobs'는 기존 10분 자동수집용입니다.
- local-agent는 웹의 즉시수집 요청을 받기 위한 별도 프로그램입니다.

적용
1. 프로젝트 폴더에 전부 덮어쓰기
2. GitHub Desktop Commit → Push
3. Firebase Firestore 규칙에 firestore.rules.txt 내용을 게시
4. backend\.env.local.example을 복사하여 backend\.env.local 생성
5. .env.local에 Firebase JSON 경로와 쿠팡 키 입력
6. backend\start-local-agent.cmd 더블클릭
7. '로컬 수집기 실행 중' 문구가 뜨면 창을 계속 켜두기

보안
- backend\.env.local은 .gitignore에 포함되어 GitHub에 업로드되지 않습니다.
- API 키와 서비스 계정 키는 채팅이나 공개 저장소에 넣지 마세요.
