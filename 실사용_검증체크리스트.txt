[ ] 쿠팡·스마트스토어·11번가·롯데온의 공식 API 현재상태가 화면에 반영된다.

[ ] 같은 주문을 다시 수집해도 월 주문·매출이 중복 증가하지 않는다.
[ ] 두 번째 자동 수집부터 DB 읽기·쓰기가 0 또는 실제 변경 건수 수준이다.
[ ] 쿠팡·스마트스토어·11번가·롯데온의 다음 실제 신규주문 알림에서 상품 썸네일을 확인한다.
[ ] 썸네일을 찾지 못해도 주문 문자 알림은 정상 전송된다.
