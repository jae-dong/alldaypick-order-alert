FINAL v7.7.5 실사용 검증 체크리스트

[ ] 웹 상단 버전이 FINAL v7.7.5이다.
[ ] 검은 창에 alldaypick-order-alert@7.7.5가 표시된다.
[ ] 수집 진행률이 100%까지 올라간다.
[ ] 오늘 주문·오늘 매출이 G마켓·옥션 제외 기준으로 맞다.
[ ] 롯데온의 현재 주문이 오늘 주문·매출·발송대기에 빠짐없이 반영된다.
[ ] 완료된 쿠팡 교환은 교환요청에서 빠지고 실제 미완료 교환만 남는다.
[ ] 신규주문이 발송대기로 이동하면 신규에서 빠진다.
[ ] 배송중·배송완료가 되면 발송대기에서 빠진다.
[ ] 완료된 문의 캐시는 빠지고 실제 미완료 문의만 남는다.
[ ] 반품·교환·문의 미완료 건은 다음 수집 후에도 남는다.
[ ] 같은 주문을 다시 수집해도 월 주문·매출이 중복 증가하지 않는다.
[ ] 두 번째 자동 수집부터 DB 읽기·쓰기가 0 또는 실제 변경 건수 수준이다.
[ ] 휴대폰 홈 화면 아이콘이 아드님 사진 디자인으로 표시된다.
