올데이픽 주문알림 v56 Firebase 로딩 긴급 수정

확인된 상태
- PC 수집기 v55는 정상 실행 중
- 쿠팡, 스마트스토어, 11번가, 롯데온 수집 완료
- 화면만 Firebase SDK 불러오는 중에서 멈춤

원인
- 동적으로 Firebase SDK를 추가하는 방식이 일부 모바일 Chrome에서 완료되지 않음
- Firestore를 읽지 못해 주문, 연결 현황, PC 상태가 모두 0 또는 확인 중으로 표시됨

수정
- 동적 Firebase 로더 제거
- 검증된 Firebase 10.14.1 고정 스크립트 방식 적용
- 20초 이상 로딩이 멈추면 실제 오류 메시지 표시
- 서비스워커 캐시 강제 갱신

적용
1. 압축 해제
2. apply-v56.cmd, apply-v56.ps1을 기존 프로젝트 폴더에 복사
3. apply-v56.cmd 실행
