올데이픽 주문알림 CLEAN v2.0.0

이번 버전은 실제 원본 프로젝트를 기준으로 다시 구성했습니다.

집계 기준
- 주문상품별 최신 상태 1건만 사용
- 오늘 주문/매출은 주문번호별로 합산
- 처리알림/쇼핑몰별 현황/미완료 목록은 동일한 미완료 원본
- 완료·배송완료·구매확정·처리완료 상태 자동 제외
- 주문통계도 동일한 정상 주문 원본 사용

적용
1. ALLDAYPICK_CLEAN_2_0_0 폴더를 기존 프로젝트 폴더 안에 넣기
2. INSTALL_CLEAN_2_0_0.cmd 실행
3. GitHub Desktop Summary: CLEAN v2.0.0 원본재구성
4. Commit → Push
5. 휴대폰 사이트 데이터 삭제 후 재접속
6. PC 수집기는 Ctrl+C 후 backend에서 npm install, npm run agent
