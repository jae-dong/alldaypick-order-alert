올데이픽 주문알림 CLEAN v3.0.0

핵심 수정
- 주문상품별 최신 상태만 유지
- 취소/반품/교환 요청이 있으면 기존 신규/발송대기에서 자동 제외
- 처리알림, 쇼핑몰별 현황, 미완료 목록이 같은 원본 사용
- 오늘 주문/매출과 오늘분석이 같은 정상 주문 그룹 사용

적용
1. ALLDAYPICK_CLEAN_3_0_0 폴더를 프로젝트 폴더 안에 넣기
2. INSTALL_CLEAN_3_0_0.cmd 실행
3. GitHub Desktop Summary: CLEAN v3.0.0 최신상태통합
4. Commit → Push
5. 휴대폰 사이트 데이터 삭제 후 재접속
6. PC 수집기 Ctrl+C 후 backend에서 npm install, npm run agent
